# 📊 AI-Powered E-Commerce Sales Analytics Dashboard

An advanced E-Commerce Sales Analytics Dashboard built using **Python, Streamlit, Pandas, Plotly, and Machine Learning**. This project helps businesses analyze sales performance, customer behavior, product trends, and revenue growth through interactive dashboards and AI-powered insights.

---

## 🚀 Project Overview

The dashboard provides real-time analytical insights from e-commerce sales data and enables users to:

* Monitor sales performance
* Analyze customer behavior
* Identify top-performing products
* Forecast future sales using Machine Learning
* Segment customers using AI techniques
* Perform RFM Analysis
* Track business growth metrics
* Export reports in CSV and Excel format

---

## ✨ Key Features

### 📈 Sales Analytics

* Revenue Analysis
* Profit Analysis
* Order Analysis
* Average Order Value (AOV)
* Profit Margin Tracking

### 📊 Interactive Visualizations

Automatically generates business insights such as:

* Most Used Payment Method

### 📈 Sales Forecasting (Machine Learning)

* Historical Revenue Trend Analysis
* Next Month Revenue Prediction

## ⚙️ Installation Guide

### 1️⃣ Clone the Repository
